Clawlendar
GitHub PyPI Docs
MCP + API + CLI

Timestamp-first perpetual calendar for AI agents.

Clawlendar gives OpenClaw and other agent runtimes one stable time model, then projects to East/West calendars with a deterministic JSON contract.

View Repository Install from PyPI
中文摘要

先用时间戳统一绝对时间，再投影到不同历法。Clawlendar 很适合多代理系统（含 OpenClaw）做排程、时序事件与跨地区历法互通。

3 Interfaces: MCP server, FastAPI, CLI
9+ Calendars supported in core/optional adapters
JSON Contract-first for agent-to-agent integration

External Time Wheel Flow

1. Input Instanttimestamp / timestamp_ms / iso / local datetime
2. Normalizeconvert to canonical UTC instant
3. Date Basisselect local or UTC projection date
4. Bridgecanonical Gregorian date as shared pivot
5. Projecttarget calendar payloads with warnings and flags
Quick Start
pip install clawlendar

# MCP server entry
clawlendar

# FastAPI mode
pip install "clawlendar[api]"
./scripts/run_api.sh
Timeline API Example
curl -sS -X POST http://127.0.0.1:8000/timeline \
  -H "Content-Type: application/json" \
  -d '{
    "input_payload": {"timestamp": 1773014400},
    "timezone": "Asia/Taipei",
    "date_basis": "local",
    "targets": ["minguo", "japanese_era", "sexagenary", "solar_term_24"]
  }'
