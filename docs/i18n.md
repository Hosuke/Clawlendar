# I18N and Locale Behavior

## Supported locale tags

- `en`
- `zh`
- `zh-CN` (normalized to `zh-Hans`)
- `zh-TW` (normalized to `zh-Hant`)
- `zh-Hans`
- `zh-Hant`

## Localized targets

- `sexagenary` payload adds localized `stem_label`, `branch_label`, and `display`.
- `solar_term_24` payload adds localized term labels and `display`.
- `day_profile` metaphysics uses locale-sensitive labels where available.

## Chinese lunar textual labels

When `lunar_python` is installed and `include_metaphysics=true`:

- `day_profile.metaphysics.eastern.lunar_date.month_name`
- `day_profile.metaphysics.eastern.lunar_date.day_name`

Example values: `正`, `二`, `十五`, `廿一`.

## Fallback behavior

- If optional providers are missing, Clawlendar returns warnings and fallback approximations where implemented.
- Machine-readable fields remain stable even when localized display labels are unavailable.
